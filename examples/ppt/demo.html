Markslide

文档转换与预览工具

作者：Markslide 团队

功能特点

  • 支持多种文件格式
  • 实时预览和编辑
  • 响应式设计
  • 深色模式支持

代码展示

function createSlide(content) {
  return {
    title: content.title,
    content: content.body,
    style: {
      theme: 'default',
      transition: 'slide'
    }
  };
}
